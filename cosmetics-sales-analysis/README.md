
# 化妆品贸易公司销售数据分析

## 项目背景
本项目基于真实贸易公司的销售数据，分析各渠道（批发、抖音、快手、得物）
的盈利情况，帮助优化销售策略。

## 使用技术
- Python 3
- pandas（数据清洗、分组分析、跨表匹配）
- matplotlib（数据可视化）
- Google Colab（开发环境）

## 数据来源
- 批发销售流水
- 零售代发账单（抖音/快手/得物）
- 平台佣金账单（抖音/快手官方导出）

## 主要分析内容
1. 各渠道销售额和利润对比
2. 平台佣金精确匹配（订单号关联抖音/快手账单）
3. 扣除佣金和平台服务费后的实际利润
4. 月度销售趋势分析
5. 退货率分析
6. 地区订单分布（Top10省份）
7. 月度自动化报告模板

## 核心发现
- 2026年1-4月合计总利润：32.6万元
- 批发渠道贡献99.5%利润（31.1万元）
- 抖音精选联盟佣金率高达90%，实际每单利润仅1.4元
- 零售代发34万销售额实际利润仅1.5万（4.4%）
- 得物平台无佣金，利润最干净
- 广东省为最大零售市场（17单）
