
# 化妆品贸易公司销售数据分析

## 项目背景
本项目基于真实贸易公司的销售数据，分析各电商平台（抖音、快手、得物）
及线下渠道的盈利情况，帮助优化销售策略。

## 使用技术
- Python 3
- pandas（数据清洗、分组分析）
- matplotlib（数据可视化）
- Google Colab（开发环境）

## 主要分析内容
1. 各平台销售量和利润对比
2. 扣除佣金和平台服务费后的实际利润
3. 退货率分析
4. 地区订单分布（Top10省份）
5. 月度自动化佣金匹配模板

## 核心发现
- 线下批发渠道利润是电商渠道总和的10倍
- 抖音精选联盟佣金率高达90%，实际每单利润仅1.4元
- 得物平台无佣金，利润最干净
- 广东省为最大零售市场（17单）

## 数据说明
真实数据已脱敏处理，示例数据结构见 sample_data.csv
